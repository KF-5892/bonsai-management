## 前提: 既存デザインの設計原則（厳守）

既に作成済みのデザイン案（`stitch_20260402/`）と同一のトーン＆マナーを維持してください。

- **DESIGN.md の "Evergreen Moss" テーマに準拠**
- カラー: Primary `#0d631b`、Surface `#f6faf6`、Surface Container Low `#f1f5f1`
- **No-Line Rule**: 1pxボーダー禁止。背景色の段差で領域を区切る
- **Glassmorphism**: ボトムナビ・FABに `backdrop-blur: 20px` + 80%透過
- **Botanical Gradient**: CTAに `#0d631b` → `#2e7d32` のグラデーション
- **Tonal Layering**: ドロップシャドウではなく背景色の明暗で深度を表現
- フォント: Inter（見出し・数値）/ Noto Sans JP（本文）
- 角丸: 12px（カード）、Full round（ボタン・チップ）
- **ボトムナビゲーション**: 全画面共通（ホーム / スケジュール / ログ記録 / お役立ち）
- スマートフォン幅（375px〜414px）前提

### 画面B: 盆栽の新規登録 / 編集フォーム

```

## 画面構成

- ヘッダー: ← 戻る + 「盆栽を登録」（編集時は「盆栽を編集」）+ 右上「保存する」ボタン
- セクション構成（Surface Nesting でセクション区切り）:

- 大きめの写真エリア（タップでカメラ/アルバム選択）

### セクション3: 入手情報
- 入手日（日付ピッカー）
- 入手方法（チップ選択）: 購入 / 頂き物 / 実生 / 挿し木 / その他

### セクション4: 管理情報

- 用土情報（テキスト入力）— 例:「赤玉土7:桐生砂3」
- 鉢情報（テキスト入力）— 例:「丸鉢 30cm」
- 体調ステータス（チップ選択）: 良好 / 要注意 / 不調

### セクション5: タグ・メモ
- タグ（既存タグから選択 + 新規作成）— チップ形式で複数選択可
- メモ（複数行テキスト）

### 最下部
- 「登録する」ボタン（Primary、Full width、Botanical Gradient）
- 編集時は「削除する」リンク（テキストのみ、Error色）

## データ参照
- bonsai_plants テーブル: name, species_id, estimated_age, acquired_date,
  acquired_method, location, soil_info, pot_info, health_status, note, main_image_url
- tags / bonsai_tags テーブル

### 画面F: 年間スケジュール（モバイル版）

```
年間スケジュールのモバイル表示を作成してください。
これは本アプリの差別化ポイントとなる重要画面です。

## 画面構成

- ヘッダー: 「年間スケジュール」+ 年表示「2026年」+ 前年/翌年切替
- フィルタ: 盆栽選択（「全て」/ 個別の盆栽名）

### 表示方式: 月カード縦リスト
- 各月をカードとして縦に並べる（1月〜12月）
- 各カード内:
  - 月名（大きめ表示、例:「3月」）
  - 作業チップの一覧（色分け）:
    - 潅水: 緑チップ
    - 施肥: 黄チップ
    - 剪定: ピンクチップ
    - 植え替え: 紫チップ
    - 観察: グレーチップ
  - 各チップをタップで詳細展開（対象の盆栽名、具体的な作業内容）
- 現在の月はアクセントカラーのボーダーまたは背景色で強調
- 過去の月はやや薄い表示

### 代替案: 横スクロールタイムライン
- 画面上部に12ヶ月の横スクロールバー（現在月にスナップ）
- 選択月の作業リストを下部に表示
- 盆栽ごとに行をグルーピング

## 設計意図
- デスクトップ版ではガントチャート/表形式で表示する想定だが、
  モバイルでは縦スクロールで情報を消化できる形にする
- 「品種に紐づく年間スケジュールで時期を一目で把握」が差別化ポイント

## データ参照
- care_schedules テーブル: repeat_type, repeat_rule, next_run_at, task_type
- bonsai_plants テーブル: name, species_id
- bonsai_species テーブル: name_ja, pruning_season, repotting_season 等の適期情報
