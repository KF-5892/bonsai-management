# データ項目定義書（現行スキーマ準拠）

- 生成日: 2026-06-27
- **出典**: `apps/*/models.py`（実装の真実 = source of truth）を 1:1 で反映。
- 位置づけ: 本書が**現行スキーマの正式な定義書**。
  実装前のたたき台 [データ項目定義調査結果.md](../データ項目定義調査結果.md) は初期検討アーカイブ（実装と乖離あり）。
  不足項目の検討は [データ項目追加検討_Stitch反映.md](データ項目追加検討_Stitch反映.md) を参照。
- 凡例（必須列）: ○=必須 / △=任意（`null`/`blank` 許容）/ PK=主キー（自動） / Auto=自動設定。
- 共通仕様:
  - 主キー `id` は全モデル **UUIDv7**（`CharField(36)`, `editable=False`, `apps.common.utils.uuid7_str`）。
  - `created_at`=`auto_now_add` / `updated_at`=`auto_now`（記載のあるモデルのみ）。
  - 「英名」= モデルのフィールド名（DB カラム名）。

## 付録A. 選択肢（TextChoices / IntegerChoices）一覧

### TaskType（作業種別・15種） — bonsai / schedules / logs で共有
`watering`潅水 / `fertilizing`施肥 / `pruning`剪定 / `repotting`植え替え / `observation`観察 /
`bud_pinching`芽摘み / `defoliation`葉刈り / `leaf_misting`葉水 / `wiring`針金かけ / `unwiring`針金外し /
`pest_control`消毒 / `jin_shari`ジン・シャリ作り / `air_layering`取り木 / `cutting`挿し木 / `other`その他

> 注: `apps.bonsai.models.TaskType` と `apps.schedules.models.TaskType` は循環 import 回避のため**二重定義**。値は一致必須（変更時は両方同期）。

### SpeciesCategory（品種カテゴリ）
`conifer`針葉樹 / `broadleaf`広葉樹 / `flowering`花物 / `fruiting`実物 / `other`その他

### HealthStatus（個体の健康状態）
`good`良好 / `watch`要注意 / `poor`不調

### FertilizerForm（肥料の形状）
`solid`固形 / `liquid`液体 / `other`その他

### Weather（作業時の天候）
`sunny`晴れ / `cloudy`曇り / `rainy`雨 / `snowy`雪 / `other`その他

### HealthEvaluation（5段階状態評価・IntegerChoices）
`1`非常に悪い / `2`悪い / `3`普通 / `4`良い / `5`非常に良い

### RepeatType（繰り返し種別）
`none`繰り返しなし / `daily`毎日 / `weekly`毎週 / `monthly`毎月 / `yearly`毎年 / `custom`カスタム

### CompletionSourceType（完了の参照元）
`species_task`品種マスタ由来 / `schedule`ユーザー個別スケジュール / `advice`月次共通アドバイス

### CompletionStatus（完了状態）
`done`完了 / `skipped`スキップ / `pending`保留

### Role（ユーザー役割）
`user`一般ユーザー / `admin`管理者

### ArticleStatus（記事公開ステータス）
`draft`下書き / `published`公開

## 付録B. エンティティ関連（概略）

```
User ─1:N─ BonsaiPlant ─1:N─ CareLog ─N:1─ Fertilizer
            │  ├─1:N─ BonsaiMedia (cover_media で 1 件を代表参照)
            │  ├─M:N─ Tag (through BonsaiTag)
            │  ├─1:N─ CareSchedule
            │  └─N:1─ BonsaiSpecies ─M:N─ HelpArticle (through ArticleSpeciesRelation)
User ─1:N─ CareScheduleCompletion ─N:1─ CareLog (任意)
User ─1:N─ Fertilizer (user=NULL は共通マスタ)
MonthlyAdvice（全ユーザー共通・FK なし）
```
