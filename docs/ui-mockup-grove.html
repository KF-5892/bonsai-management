設計メモ

  • No-Lineルール：1px borderの代わりに、surface tier の重なり（surface-low #f1f5f1 / lowest #ffffff）で領域を区切る。
  • Tonal Layering：濃いベース（#f6faf6）の上に薄いコンテナ（#f1f5f1）を敷き、その上に白のカード（#ffffff）を配置して立体感を演出。
  • ボタニカルグラデーション：FAB / 主要CTA / Heroで linear-gradient(135deg, #0d631b, #2e7d32)。
  • ガラスモーフィズム：トップバー / ボトムナビは rgba(246,250,246,0.78) + blur(20px)。
  • アクセントバー：盆栽カード左端 4px。状態色（健康=緑 / 要注意=tertiary / 注意=オレンジ）で識別。
  • タイポ：英字 Inter / 日本語 Noto Sans JP。Display は -0.02em のトラッキングで編集的に。
