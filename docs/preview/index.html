🌳 盆栽管理アプリ — 実装UIプレビュー

Phase 1（MVP）の Django 実装画面を、デモデータ付きで静的書き出ししたものです。

⚠️ これは閲覧専用の静的スナップショットです。フォーム送信・検索・HTMX などの 動的操作は動作しません（リンクの一部は無効化しています）。スタイル（Tailwind 等）は CDN 経由で読み込むため、オンライン環境のブラウザで正しく表示されます。
