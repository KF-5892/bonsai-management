# モバイルUIモックアップ作成 — AI依頼プロンプト

以下のプロンプトをAI（画像生成AI、またはUIモック生成AI）に渡してください。

---

## プロンプト本文

```
盆栽管理Webアプリのモバイル（スマートフォン）向けUIモックアップを作成してください。

## アプリ概要

盆栽愛好家が自分の盆栽を管理するためのWebアプリです。
主なユーザーは盆栽の専門家〜中級者で、屋外（庭先・ベランダ）でスマートフォンを片手に操作する場面が多いです。

## デザインシステム

- Material Design 3（M3）に準拠
- カラースキーム: 緑系（盆栽・植物のドメインに合わせた自然な配色）
  - Primary: #2E7D32（濃い緑）
  - Primary Container: #B9F6CA（薄い緑）
  - Surface: #F7FBF7（ほぼ白の緑がかった背景）
  - Error: #BA1A1A
- フォント: Roboto / system-ui
- 角丸: 12px（カード）、Full round（ボタン・バッジ）
- エレベーション: M3 の elevation 1〜3
- 状態バッジは色＋アイコンで区別（アクセシビリティ対応）
  - 良好: 緑背景 + ✓ アイコン
  - 要注意: ローズ背景 + ⚠ アイコン
  - 不調: 赤背景 + ✕ アイコン

## 作成してほしい画面（計7画面）

### 画面1: マイ盆栽一覧（ホーム）
- 上部: アプリロゴ「盆栽管理」+ 通知ベルアイコン + ユーザーアバター
- 検索バー（キーワード検索）
- フィルタチップ: すべて / 良好 / 要注意 / 不調
- 盆栽カードのリスト（縦スクロール）:
  - 各カード: 代表写真（左サムネイル）、盆栽名、品種名、体調バッジ（✓良好 / ⚠要注意）、次の作業（例:「水やり 今日」「剪定 3/25」）
  - 寄せ植えグループはグループカード＋内部にメンバーリストを折りたたみ表示
- 右下: FAB（＋ボタン）「新規登録」
- 下部: ボトムナビゲーション（ホーム / スケジュール / ログ記録 / お役立ち）

### 画面2: 盆栽詳細
- ヘッダー: ← 戻るボタン + 盆栽名「黒松 太郎」 + 編集ボタン
- 代表写真（大きめ、横幅いっぱい、タップで拡大）
- 基本情報カード:
  - 品種: 黒松（Pinus thunbergii）
  - 体調: ✓ 良好（緑バッジ）
  - 入手日: 2020/04/15
  - 推定樹齢: 42年
  - 設置場所: ベランダ東側
  - 用土: 赤玉土7:桐生砂3
  - 鉢: 丸鉢 30cm
- タグ: 「展示会用」「⚠ 要注意」
- セクションタブ: 作業ログ / スケジュール / メディア / 植え替え履歴
- 作業ログセクション（デフォルト表示）:
  - 直近のログ3〜4件を時系列で表示（日付、作業種別チップ、メモ抜粋、天気アイコン、気温）
  - 「すべてのログを見る」リンク

### 画面3: 作業ログ記録（入力フォーム）
- ヘッダー: ← 戻る + 「作業を記録」
- フォーム項目:
  - 対象の盆栽（選択済み or ドロップダウン）
  - 作業種別（チップ選択式）: 潅水 / 施肥 / 剪定 / 植え替え / 観察 / 芽摘み / 葉刈り / 葉水 / 針金かけ / 針金外し / 消毒 / ジン・シャリ作り / 取り木 / 挿し木 / その他
  - 実施日時（日付ピッカー）
  - 天候（アイコン付き選択: ☀晴れ / ☁曇り / 🌧雨 / ❄雪 / その他）
  - 気温（数値入力 + ℃）
  - メモ（複数行テキスト）
  - 写真添付（複数可、カメラ撮影 or アルバムから選択）
  - 評価（★1〜5の星レーティング）
- 「記録する」ボタン（Primary、Full width）

### 画面4: 月別スケジュール
- ヘッダー: 「スケジュール」+ 年月表示「2026年3月」
- 月切替: < 前月 | 今月 | 次月 >
- 最上段: 共通ルールカード（水色背景、左に青いボーダーライン）
  - タイトル: 「共通ルール（3月）」
  - 内容: 水やり頻度（例: 1日1回）、季節メモ（例: 西日回避解除時期）
- その下に品種ごとのカードを縦リスト:
  - 各カード: 植物名、作業チップ（色付き: 変化=紫, 手入れ=ピンク, 肥料=黄, 水やり=緑）、簡単な説明テキスト
  - 寄せ植えグループは見出し付きでまとめて表示
- 右上にカレンダーアイコン（年間スケジュールへの切替リンク）

### 画面5: 植え替え履歴・サマリー
- ヘッダー: ← 戻る + 「植え替え履歴」
- アラートバナー（植え替え時期が近い場合）: ⚠「次回植え替え時期です」
- サマリーカード:
  - 最終植え替え: 2023/03/15
  - 次回目安: 2025年春
  - 用土配合: 赤玉土7:桐生砂3
- 履歴リスト（時系列降順）:
  - 各行: 日付、用土配合、鉢情報、メモ

### 画面6: メディア（写真）タイムライン
- ヘッダー: ← 戻る + 「メディア」
- 年月ごとにグルーピングされた写真グリッド（2列）
- 各写真にキャプション（日付 + 種別チップ）
- 上部に年フィルタ（2024 / 2025 / 2026）
- 「去年の今頃」セクション: 前年同月の写真を横スクロールカルーセルで表示（成長比較用）

### 画面7: 空状態（初回起動・盆栽0本）
- 中央配置のイラスト（盆栽のシンプルなアイコン or イラスト）
- テキスト: 「まだ盆栽が登録されていません」
- サブテキスト: 「最初の盆栽を登録して、年間の手入れ予定を管理しましょう」
- CTA ボタン: 「盆栽を登録する」（Primary、大きめ）
- 下部リンク: 「お役立ち記事を読む →」

## デザイン上の注意事項

- スマートフォン幅（375px〜414px）を想定
- 片手操作を意識し、重要なアクション（ログ記録、新規登録）は画面下部に配置
- 屋外での視認性を考慮し、コントラストは高めに
- 写真を多用するアプリのため、画像の表示領域は十分に確保
- ボトムナビゲーションは全画面共通で表示
- 日本語テキストで作成
```

## 補足: 生成AI別の使い分け

| ツール | 用途 | 備考 |
|--------|------|------|
| v0 (Vercel) | インタラクティブなHTMLモック | React + Tailwind で実装コード付き |
| Claude Artifacts | 単一HTMLモック | プロンプトそのまま使用可 |
| Figma AI / Galileo AI | 高精細なデザインモック | 画面ごとに分割して依頼すると良い |
| Midjourney / DALL-E | コンセプトイメージ | 「UI design, mobile app, bonsai management」等の英語キーワード併用 |

画面数が多い場合は、まず **画面1（一覧）と画面4（月別スケジュール）** を優先して作成し、トーン＆マナーを確認してから残りの画面に進むことを推奨します。
