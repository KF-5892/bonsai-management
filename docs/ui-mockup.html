凡例（作業種別と色の対応）

  • 先頭の水色エリアは共通ルール（水やり・西日・軒下）。品種に依存しないため1か所に集約しています。夕方の葉水は要否が品種で異なるため共通帯には含めません。
  • 変化（開花・落葉・芽吹きなど）：紫系。品種ごとに異なります。
  • 水やり：緑系。本モックでは共通ルール帯のみに表示し、植物行では重複しません。
  • 手入れ：ピンク系。西日・軒下は共通帯、剪定・植え替え・夕方の葉水などは品種行に記載します。
  • 肥料：黄系。品種ごとの施肥月（セル表の肥料バーと同義）。

実装時は植物単位で行をまとめて折りたたみ可能にすると一覧が短くなります。3月列の強調は「今月」デモ用です。

このモックで対応済みの改善

  • 改善 1.1：モバイルスケジュール表示 - 月タブ＋カード縦リストによる片手操作向け代替ビューを追加。
  • 改善 1.2：ナビゲーション改善 - ヘッダーに詳細（モック）、モバイル、初回、お役立ちリンクを追加。

  • 改善 1.4：ステータスバッジのアクセシビリティ - 良好（✓）、要注意（⚠）、不調（✗）のテキストアイコンを全箇所に追加。色だけでなく形状でも判別可能に。
  • 改善 1.5：空状態画面 - イラストエリア、CTA、お役立ち記事リンクを追加。
